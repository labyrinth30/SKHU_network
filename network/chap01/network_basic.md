## 1) 네트워크(Network)
네트워크는 서로 데이터를 주고 받을 수 있는 컴퓨터 및 장치들의 집합이다

### 노드
네트워크 안 각각의 장비(라우터, 브리지, 게이트웨이, 컴퓨터 등)

바이트 순서에는 big-endian과 little-endian이 있다.
이진수를 저장하거나 전송할 때, big-endian에서는 큰 자리부터 8bit씩 저장(전송)한다.
little-endian에서는 작은 자리부터 8bit씩 저장(전송)한다.

### 네트워크 주소의 종류
이더넷 네트워크에서 주소는 MAC 주소.
IP 네트워크에서 주소는 IP 주소.
가장 대표적 ip 네트워크 == 인터넷
즉, 인터넷 주소는 ip 주소.

### 주소에 대한 이름
어떤 노드 이름 하나는 어떤 노드 주소 하나를 가르킨다.

### 패킷(packet)
컴퓨터 네트워크는 패킷 교환(packet-switched) 방식을 사용한다.
데이터를 덩어리로 쪼개어 전송되고, 각각의 패킷에는 송수신자의 정보가 포함됨.
##### 장점
회선 하나에 여러 패킷 공유 가능
전송과정에서 손상된 패킷만 체크섬으로 검사하여 재전송 가능함.

### 프로토콜
1. HTTP - 웹 서버와 브라우저 간의 통신 방법
2. IEEE 802.3 표준 - 특정 유형의 회선에서 비트를 전기 신호로 인코딩하는 방법

### 계층형 아키텍쳐
몇 개의 계층으로 구분되어 설계 구현된 시스템
##### 장점
각 계층은 인접한 계층하고만 대화하기 때문에 개발자가 전체 시스템을 파악할 필요x
인접한 계층 사이의 인터페이스가 변경되지 않는 한, 계층 안에 있는 소프트웨어를 타 계층에 미치는 영향 없이 수정하거나 대체할 수 있다.





# 연습 문제

Ethernet 주소와 IP 주소의 차이는 무엇인가?  
답: 이더넷 네트워크의 주소는 MAC 주소이고, IP는 인터넷 네트워크의 주소이다.

TCP/IP 4 계층은?  
답: 호스트-투-네트워크 계층/인터넷 계층/전송 계층/응용 계층

TCP/IP 4 계층을 위에서부터 순서대로 적어라.  
답:애플리케이션 계층 / 전송 계층 / 인터넷 계층 / 호스트-투-네트워크 계층

TCP/IP 4 계층 각각의 역할은?  
답: 
호스트 투 네트워크 계층 = 인터넷 계층으로부터 받은 ip 데이터그램을 적합한 물리적 신호로 변환하여 전송
인터넷 계층 = 전송 계층으로부터 받은 데이터를 ip 데이터그램으로 쪼개어 호스트-투-네트워크층으로 전송, 라우팅 알고리즘 정의
전송 계층 = 보내는 모든 데이터가 손실되지 않음, 손상되지 않음, 순서 보장함.

TCP/IP 4 계층에서 각 계층의 역할을 간단히 설명하라.  
답:

라우터 장비의 역할은 무엇인가?  
답: 패킷의 목적지 주소를 보고 최적의 경로를 계산하여 전달함.

DNS란 무엇인가?  
답: 숫자로 된 인터넷 주소를 사람이 기억할 수 있는 호스트네임으로 변환하는 시스템

DHCP란 무엇인가?  
답:  동적 호스트 구성 프로토콜은 서버와 달리 lan이나 무선에 연결된 장비는 DHCP 를 통해 ip 주소를 할당 받는다.

컴퓨터 자기 자신을 가르키는 IP 주소는?  
답:   루프백 주소 127.0.0.1

포트 번호
답: 컴퓨터에서 실행되는 여러 애플리케이션을 구별하는 번호

IP 주소에 더불어 포트 번호가 필요한 이유는 무엇인가?  
답:  같은 ip 주소에서 실행되는 여러 통신 앱을 구분하기 위해서

성공회대학교는 KT ISP로부터 아래와 같은 internet 주소 블럭을 할당 받았다. 203.246.75.0 ~ 203.246.75.255 (/24)  
/24 부분의 의미는 무엇인가?
답: 호스트 네임과 네트워크 네임을 구분하는 용도, 203.246.75.0 ~ 255까지 256개의 호스트 주소를 갖게 됨.

성공회대학교에서 사용 가능한 IP 주소의 수는 몇 개인가?  
답:  254

NAT 장비의 역할은 무엇인가?  
답:  네트워크 주소 변환기는 내부에서 사용하는 내부 네트워크 ip를 public ip 주소로 변환하는 용도

ethernet 네트워크에서 주소를 무엇이라고 부르는가?   
답: MAC 주소

인터넷에서 주소를 무엇이라고 부르는가?  
답 : IP 주소  

인터넷에서 데이터를 작게 나누어 전송하는, 그 작은 데이터 덩어리를 무엇이라고 부르는가?  
답 : 패킷   

컴퓨터들이 서로 대화(통신)하는 방법에 대한 자세한 규칙이고 약속을 무엇이라고 부르는가?  
답 : 프로토콜

전송 계층의 대표적인 프로토콜 두 개는 무엇인가?  
답 : TCP, UDP  

TCP/IP 4 계층 중에서 인터넷 주소가 정의된 계층은?  
답 : 인터넷 계층  

TCP/IP 4 계층 중에서 안전하고 확신할 전송 방법이 정의된 계층은?  
답 : 전송 계층  

TCP/IP 4 계층 중에서 데이터를 목적지까지 전달하는 방법이 정의된 계층은?  
답 : 전송 계층

IP 부족 문제를 해결하기 위한, 내부 네트워크 주소와 외부 네트워크 주소의 변환 기술을 무엇이라고 부르는가?  
답 : NAT

203.246.75.0 ~ 203.246.75.255 (/24) 에서 /24 부분의 의미는 무엇인가?  
답 : 네트워크 주소와 호스트 주소를 구분하는 용도

파일에서 문자열을 읽어오는 기능이 구현된 클래스는?  
답 : FileInputStream

객체를 byte[] 데이터로 변환하는 것을 무엇이라고 부르는가?  
답 : 인코딩
